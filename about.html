What is rubber stamp?

Handmaded Stamp by Using Rubber.

Rubber Stamp is handmaded stamp by people. It is using hobby knife engrave rubber to produce a seal. It's function same as stamp which we usually see in daily life.Stamp pattern in the paper. However, the special things is Rubber Stamp's pattern can be more flexible. It is because every partten like characters or words can be produce if you have good technique. Because of the flexibility,it is very useful that Rubber Stamp can be used in diverse way. Making a Rubber Stamp is really time taking, therefore it can say that a dedicate Rubber Stamp is art work.

Why I start to make rubber stamp?

Rubber Stamp is Part of My Life.

When I was secondary student,rubber stamp become popular. I saw some beautiful and awesome rubber stamp in Instagram. And I was very curious about how they can produce those stamp.Then I started watching YouTube to learn how to do it.I remember that my first stamp is just using simple tools -cutter and eraser to finish a stamp which is an animation character face.After that I was interested on it.Therefore, I have hold an Instragram page for share my work to others.And Through making rubber stamp,I have trained become more pation.
